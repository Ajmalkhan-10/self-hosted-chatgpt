# self-hosted-chatgpt
Self Hosted ChatGPT

![Screenshot](.images/image.png)

## Installation

Pre-requisites:
- Docker

```bash
docker compose up -d
```

## Running

- Navigate to `http://localhost:8080`

---

## Customization

- This uses `qwen3:8b` LLM model using ollama, feel free to change this if needed
- Uses SearXNG for Searching on Internet
